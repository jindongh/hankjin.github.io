Mokee Usage and Tips Samsung SC-03E

Install Mokee on sc-03e Step1: install custom recovery

  • download Odin3 from Here
  • download recovery 6.0.2.9 from Here
  • Connect phone to Windows with USB
  • Press Key(downside of left side), Key(right side) and Key(bottom) at the same time until the phone reboot, and then release
  • Press Key(upside of left side) once to enter flush mode
  • extrace odin3.rar to a folder, and run Odin3.exe
  • Click PDA button on Odin3 and select recovery-6.0.2.9-sc03e.tar downloaded
  • Click Start button on Odin3 to beginn flush recovery
  • OK

Install Mokee on sc-03e Step2: install custom recovery

  • download recovery_CWM_6.0.4.5.zip from Here
  • download sc03e_root.zip from Here
  • download mokee from Here
  • copy the downloaded recovery_CWM_6.0.4.5.zip, sc03e_root.zip, and mokee rom to the phone with USB
  • Press Key(upside of left side), Key(right side) and Key(bottom) at the same time until the phone reboot, and then release buttons
  • using and to select install zip, then using button to enter
  • select choose zip from /sdcard and enter
  • select the recovery_CWM_6.0.4.5.zip and enter
  • select Yes- Install
  • after install, reboot the machine

Install Mokee on sc-03e Step3: root

  • Press Key(upside of left side), Key(right side) and Key(bottom) at the same time until the phone reboot, and then release buttons
  • using and to select install zip, then using button to enter
  • select choose zip from /sdcard and enter
  • select the sc03e_root.zip and enter
  • select Yes- Install
  • after install, reboot the machine, and it is ok

    • Install Mokee on sc-03e Step4: install mokee

  • Press Key(upside of left side), Key(right side) and Key(bottom) at the same time until the phone reboot, and then release buttons
  • using and to select install zip, then using button to enter
  • select choose zip from /sdcard and enter
  • select the mokee zip and enter
  • select Yes- Install
  • after install, reboot the machine, and it is ok

    • Tip 1:mokee install Error: This package is for m3,m3xx,sc03e,SC-03E,GT-I9305,i9305 devices; this is a SC03E

    Solution: open the mokee rom MK44.2-sc03e.zip\META-INF\com\google\android\updater-script and remove the first lines:

    assert(getprop("ro.product.device") == "m3" || getprop("ro.build.product") == "m3" || 
       getprop("ro.product.device") == "m3xx" || getprop("ro.build.product") == "m3xx" || 
       getprop("ro.product.device") == "sc03e" || getprop("ro.build.product") == "sc03e" || 
       getprop("ro.product.device") == "SC-03E" || getprop("ro.build.product") == "SC-03E" || 
       getprop("ro.product.device") == "GT-I9305" || getprop("ro.build.product") == "GT-I9305" || 
       getprop("ro.product.device") == "i9305" || getprop("ro.build.product") == "i9305" || abort("This package is for \"m3,m3xx,sc03e,SC-03E,GT-I9305,i9305\" devices; this is a \"" + getprop("ro.product.device") + "\"."););

    and then remove the file MK44.2-sc03e.zip\META_INF\CERT.SF and META_INF\CERT.RSA

    Using APKSign to resign the zip

    copy the zip file back to phone, and retry to install with recovery

    Tip 2:mokee upgrade Error: Versions mismatch...

    Solution: upgrade the old updates before update the latest, it is increasingly upgrade so it is wrong to upgrade without older version

    Tip 3:how to restore to original state for sc-03e

    after serveral operation, the phone is totally out of control, or regret for the determination, how to restore to original state?

    just download the original version from samsung SC03EOMAMB1_SC03EDCMAMB1_SC03EOMAMB1_HOME_REV00_user_low_true.tar from Here, and using Odin3 to flush it
