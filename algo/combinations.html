Algorithm/Combinations

Question

how to iterate combinations of an array

eg: [1,3,4]

  1. [1], [3], [4]
  2. [1,3] [1,4] [3,4]
  3. [1,3,4]

Solution

usually we state a combination by which elements appear eg 3 appear, on the other hand, we can state it this way: n-th appear or not appear eg 2nd appear

and we use 0/1 mark appear or not, that is the methods

  1. [1, 3, 4]
  2. [1, 0, 0] => [1]
  3. [0, 1, 0] => [3]
  4. [0, 0, 1] => [4]
  5. [1, 1, 0] => [1,3]
  6. [1, 0, 1] => [1,4]
  7. [0, 1, 1] => [3,4]
  8. [1, 1, 1] => [1,3,4]
